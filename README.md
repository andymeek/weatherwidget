# WeatherWidget

## Install & Run

    npm i

To build all:

    npm run build

To start server:

    npm run start

Examples can be found in (make sure the Node modules are installed and the JS and SCSS is built):

    dist/index.html

## To add/view widgets

    Add new widgets - eg. http://localhost:3000/editor
    View the widgets - eg. http://localhost:3000/widgets
